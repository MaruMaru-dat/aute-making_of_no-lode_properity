### 【使用手順書】

1. 解析結果を以下のフォルダ構造にしてまとめる。ファイル名はAとする。
このとき、nフォルダは電圧or電流値、mフォルダは手法名(ABH,sinB and Blongなど)とし、
そのあとにjw法または過渡解析の結果を入力しておく。

2. アプリを起動してAのフォルダを指定し、データをまとめるエクセルファイルを指定する。


【フォルダの構造】
```
A
 ├─ n01
 │   ├─ m01
 │   │   └─ jw
 │   │       └─ circuit.csv
 │   └─ m02
 │       └─ ...
 └─ n02
     └─ ...
```
